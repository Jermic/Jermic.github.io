Mac 软件清单 v0.1

Mac 常用软件清单

命令行工具

  • iTerm2 - 终端工具 替代自带的 Terminal

  • Zsh - 交互式命令行 shell

  • Oh My Zsh - 用于管理 zsh 配置

  • tmux - 终端复用器

  • Termius - 终端工具 可连接远程服务器

编辑器

  • Visual Studio Code - 微软出品代码编辑器
  • Sublime Text - 高级文本编辑器
  • Vim - 上古的终端编辑器

开发工具

  • JetBrains Toolbox - JetBrains 开发全家桶
  • Xcode - 苹果出品 用于开发 iOS 和 MacOS 的开发工具
  • Sequel Pro - MySQL 数据库管理软件
  • TablePlus - 数据库管理软件, 支持多种数据
  • Navicat Premium - 数据库可视化工具
  • CodeRunner - 轻量级多语言编程编辑器
  • SwitchHosts - 切换系统 Host 环境
  • Charles - 网络调试工具
  • Sourcetree - 版本管理工具
  • Github Desktop - Github 出品版本管理工具
  • Postman - 可用于 Api 调试 模拟请求
  • Paw - 网络请求调试客户端
  • Beyond Compare - 文件对比工具
  • Dash - 强大到你无法想象的 API 离线文档软件
  • SnippetsLab - 管理和组织你的代码片段
  • Transmit - 一个 FTP 客户端

效率工具

  • Microsoft To-Do - 微软出品 TODO
  • Teambition - 团队协作 也可用作个人任务待办记录

图像处理

  • Adobe Creative Cloud - 用户安装 Adobe 全家桶

  • Sketch - 设计师必备 适用于用户界面 Web 和移动设计

  • Eagle - 设计师图片管理工具 跨平台

虚拟机

  • VirtualBox - 虚拟机

  • Mumu - 网易出品 安卓模拟器

  • Parallels Desktop - 有点贵但超级好用的 Mac 平台虚拟机

压缩工具

  • Archiver - 压缩解压缩工具

  • BetterZip 3 - 压缩解压缩工具

系统效率工具

  • Alfred 3 - 效率神器

  • PopClip - 选择文本 处理增强工具

  • Unclutter - 菜单栏增强工具集成剪贴板 文件存储 记事本

  • Bartender - 组织或隐藏 Mac 上的菜单栏图标

  • Boom 3D - 系统音效增强

  • Gestimer - 菜单栏快捷倒计时工具

  • iStat Menus - 菜单栏上的高级 Mac 系统监视器

  • Irvue - 壁纸工具

  • Karabiner - OS X 的键盘修改 增强工具

  • LilyView - 图片查看器

  • Moom - 多任务多窗口的软件

远程

  • Teamviewer - 远程桌面管理跨平台

文档影音

  • MindNode - 简洁的思维脑图

  • MWeb - 专业的 Markdown 写作软件

  • Typora - Markdown 编辑器 跨平台

  • Bear - 协作工具

  • PDF Expert - PDF 阅读、批注，编辑文本，添加照片，填写表单

  • WPS - 跨平台的办公文档处理软件

  • iina - 视频播放工具

  • 欧路词典 - 词典

  • Kindle App - 亚马逊 Kindle App 电子书阅读器

  • OmniGraffle - 可用来绘制图表 思维导图以及插图或原型

下载工具

  • Downie 3 - 视频下载工具 支持国内外很多常见视频网站

  • Dukto R6 - 局域网跨平台文件传输

  • aria2 - 一款支持多种协议的轻量级命令行下载工具

  • You-Get - 网络富媒体命令行下载工具

  • 百度网盘离线下载 - 百度网盘离线下载

  • 迅雷 - 备选下载方案 广告多

浏览器

  • Chrome Dev - 谷歌浏览器开发版

  • Firefox - 火狐浏览器

  • Safari - Mac 预装自带浏览器

网盘

  • 百度网盘 - 百度出品

  • Onedrive - 微软出品

  • 坚果云 - 口碑产品

输入法

  • 搜狗输入法 - 搜狗输入法

科学上网

  • ShadowsocksX-NG - 一款 ShadowsocksX 客户端软件
