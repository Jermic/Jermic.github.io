简介

本软件集成了 Android 开发中常见的一些编译 / 反编译工具, 方便用户对 Apk 进行逆向分析, 提供 Apk 信息查看功能. 目前主要功能包括 (详细使用方法见使用说明):

  • 反编译 APK
  • 重建 APK
  • 签名 APK
  • 优化 APK
  • DEX2JAR（APK2JAR）
  • JDGUI
  • 提取 DEX
  • 提取 XML
  • Class to smail
  • Apk 信息查看
  • Unicode 转换
